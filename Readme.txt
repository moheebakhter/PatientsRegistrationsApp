Github URL:-
https://github.com/moheebakhter/PatientsRegistrationsApp.git

How To Open:-

1. Click the link first and download the application.
2. Now open source code on your choosable platform like(e.g. Vscode).
3. Now open the terminal and go to backend folder by typing the command cd backend.
   Then type npm i command to install node modules.
   Then type npm start to run backend of application.
4. Now go to frontend folder by typing the command cd frontend.
   Then type npm i command to install node modules.
   Then type npm start to run frontend of application.     
5. Finally you can fill the form and register the patients and check all over the application.
